## Notes from my first sprites

An Aseprite Crash Course In 30 Minutes 
Steps!
Blocking in 
Silhouette 
Shading it 
Coloring it
Adding detail
Don't place pixels one at a time


Timeline
Shift n : new layer
Alt n: new frame
Painting shortcuts
B: brush
E : eraser
Alt: eye dropper
Ctrl + alt + left click: then drag right to increase brush size

Shading!
You can set a range of colors. 
Then just overwrite it to shift up one in the colors
Honestly pretty nifty!

Selections short cuts
M : marque - select a box
Color in that box m
W: select all of the same color that is touching
V: if you turn on auto select layer, I think you can just click and go to the different layers?

Tab : show/hide timeline
Control f : hide diff parts of the app
Shift s  : work you are doing will snap to the grid
